# Evolution Unfiltered Notes Feed

Minimal WordPress plugin for rendering the Git-automated `notes.json` feed.

## Install

1. Copy `eu-notes-feed` into `wp-content/plugins/`
2. Activate **Evolution Unfiltered Notes Feed**
3. Add this shortcode to the page where notes should appear:

```text
[eu_notes_feed url="https://YOUR-JSON-URL-HERE/notes.json"]
```

## Recommended shortcode

```text
[eu_notes_feed url="https://YOUR-JSON-URL-HERE/notes.json" limit="12" show_source="true" show_date="true"]
```

## Attributes

- `url`: required
- `limit`: default `12`, max `50`
- `show_source`: `true` or `false`
- `show_date`: `true` or `false`
- `empty_message`: fallback copy when there are no notes
- `cache_ttl`: cache in seconds, default `900`

## Feed source

This plugin expects the JSON generated by:

- `scripts/ingest_substack_notes.mjs`
- `.github/workflows/refresh-substack-notes.yml`

The shape should match `content/substack/data/notes.json`.

## Suggested hosting pattern

Use a stable public URL for `notes.json`, for example:

- a raw GitHub URL for a public repo
- a mirrored JSON file on your live hosting
- a CDN-backed URL

If the repo is private, do not use a private raw GitHub URL in WordPress unless you plan to proxy it.
